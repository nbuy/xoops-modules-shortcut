Shortcut module for XOOPS

What is this?

Access a WWW page bia redirect jump.

Install

Upload module `shortcut' on your XOOPS site under `modules' directory. And install it.

For more convinient, make directory under XOOPS root (XOOPS_ROOT_PATH), e.g. `sc'. And copy (or linking) from `modules/shortcut/index.php' file.

How to use

In Administration page, Register shortcut `ID' and `URL' pair.
You can access the URL via ``XOOPS_URL/sc?ID''.
Example: http://mysite.ddo.jp/sc?my It's my(auther's) contact info page.
