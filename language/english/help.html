Shortcut module for XOOPS

What is this?

Access a WWW page bia redirect jump.

Install

Upload module `shortcut' on your XOOPS site under `modules' directory. And install it.

For more convinient, copy a file modules/shortcut/sc.php to XOOPS root (XOOPS_ROOT_PATH), you can redirect jump using "Link ID" in short url. There is more smart way to use apache mod_rewrite.

How to use

Enable In Administration page, Register shortcut `ID' and `URL' pair.
You can access the URL via ``XOOPS_URL/sc?ID''.
Example: http://mysite.ddo.jp/sc?my It's my(auther's) contact info page.
